scholarship eligibility evaluator - suite de testes automatizadoseste repositorio contem a implementacao de uma suite de testes automatizados para o sistema scholarship eligibility evaluator . a atividade integra tecnicas funcionais e estruturais de geracao de testes e analise de adequacao da suite.1. contexto do projetoo sistema avalia a elegibilidade de candidatos a bolsas de estudo com base em cinco regras principais:idade: minima de 16 anos e revisao manual para menores de 18.gpa: nota minima para aprovacao direta e faixas de revisao.frequencia: taxa de presenca minima exigida.cursos obrigatorios: verificacao de conclusao de grade necessaria.registro disciplinar: checagem de historico de conduta.o sistema determina se o status do aluno e approved, rejected ou manual_review.2. escolha tecnologicalinguagem: python 3.11.9.framework de teste: pytest 9.0.2.justificativa: o pytest permite a criacao de testes parametricos, facilitando a cobertura de multiplas classes de equivalencia e valores limite com codigo limpo e eficiente.3. estrutura dos testesa suite de testes foi projetada sob duas perspectivas principais:tecnicas funcionais classes de equivalencia: identificacao de faixas validas e invalidas para gpa, frequencia e idade.analise de valor limite: testes focados nos pontos de transicao exatos das regras de negocio (ex: gpa 6.0 e 7.0).tecnicas estruturais cobertura de decisoes: exercicio de todos os desvios condicionais (if/else) relevantes.fluxos de execucao: criacao de casos que percorrem diferentes caminhos do codigo para garantir que todos os cenarios de erro sejam detectados.4. como executar os testespara reproduzir a execucao da suite, siga os passos abaixo:instalar o pytest:Bashpip install pytest
executar o comando principal:Bashpython -m pytest test_scholarship.py
5. resumo da cobertura a suite inclui os seguintes casos obrigatorios:1 caso de approved.1 caso de manual_review (com variacoes parametrizadas).5 casos de rejected por motivos distintos.2 casos de entrada invalida (validacao de gpa e frequencia).4 casos de valor limite.
